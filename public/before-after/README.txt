У цій папці лежать два знімки для сторінки «До / після»:

  before.png — ліва частина повзунка («до»)
  after.png  — права частина («після»)

Зараз це твої клінічні фото (IMG_9513 → before, IMG_9514 → after).
Якщо сторони переплутані — просто поміняй файли місцями (перейменуй)
або онови шляхи в src/config/beforeAfterAssets.js.

Для максимальної якості на сайті: поклади сюди **оригінали** (PNG без стиснення
або JPEG якість 90–100%), бажано шириною від **1600 px** — браузер сам
масштабує, але джерело має бути чітким.
